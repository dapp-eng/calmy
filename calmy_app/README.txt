Untuk menjalankan aplikasi ini, pastikan pustaka sudah terinstall dengan benar!

Buat virtual environtment jika diperlukan dengan menjalankan kode: python -m venv venv (OPSIONAL)

Lalu aktifkan virtual environtment dengan cara:
source venv/bin/activate (untuk Linux/MacOS)
venv\Scripts\activate (untuk Windows)

-- Lakukan instalasi dengan menjalankan kode berikut di terminal:

1. pip install kivy (untuk install kivy ver:2.3.0)
2. pip install https://github.com/kivymd/KivyMD/archive/master.zip (untuk install kivymd ver: 2.1.0.dev0)
3. pip install matplotlib (untuk install matplotlib)

Jika semua telah terinstall, run kode program di file calmy.py untuk mulai menjalankan aplikasi.
